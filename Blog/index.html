Thoughts/Reflections 19/12/17

Having the website is such a challenge because I don't know anything about coding... Well, I say I don't know anything, I know a lot more than I did when I began learning about Web as a medium. I understand that there's more layers than just what you see in the web browser.

I never knew that there were three separate languages you can write code in: HTML provides the bones and structure of the webpage, CSS provides the Style, and JAVASCRIPT provides the action. The three can be used together to create lovely looking webpages, but HTML will work by itself- it'll just be plain and simple.

I've picked up a copy of "Computer Arts" Magazine, and have enjoyed skim-reading it. At first glance it's a Graphic Designer's/Web Designer's magazine, with it's minimalist front cover being reflective of it's neat, well crafted interior and topics for discussion.
